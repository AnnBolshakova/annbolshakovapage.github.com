Домашняя страница

На этой странице будет размещено описание, расчет и параметризованное построение твердотельной модели в формате STL, с последующей возможностью скачивания всей модели сборки. 

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
